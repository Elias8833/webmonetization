Probabilistic Revshare Generator

This tool allows you to create a Web Monetization meta tag which splits money between multiple Payment Pointers (Learn more). If you make changes to an existing meta tag with this tool, make sure to replace the old meta tag with the updated version.

Editing Revshare

Add all Payment Pointers that will receive a split of the revenue and name them (optional). By setting weights, you determine what share each Payment Pointer will receive. You can also tweak the percentage to adjust shares.

If you want to edit an existing meta tag created with this tool, click 'Import existing revshare.' Note: Your inputs will be saved in your browser locally.
