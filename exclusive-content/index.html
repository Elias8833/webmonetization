Exclusive Content Generator

This tool allows you to create exclusive content which is only unlocked if payment can be verified. Learn more!

Generate Exclusive Content

In order to generate exclusive content, we need the following information:

Example: $spsp.example.com/alice or https://spsp.example.com/alice

This can really just be plain text, e.g. Hello World, or an HTML tag, e.g. <div><p>Hello World</p></div>

If this is not specified, we will use a hosted verifier. If you decide to use your own verifier, specify something like: https://verifier.example.com
